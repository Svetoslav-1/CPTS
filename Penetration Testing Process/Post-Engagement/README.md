# Post-Engagement

This section covers the final phase of a penetration test — delivering findings to the client, handling the report review cycle, and formally closing out the engagement.

## Hint

When a report is first handed to a client, it isn't considered final right away — the client is given a chance to review, dispute, or comment on the findings. There's a specific single-word designation for this initial version of the report, commonly used in professional and legal document workflows.
